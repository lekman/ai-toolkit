# Proportion

## Match effort to what the work is worth

Quality is relative. Perfection is the enemy of good. Calibrate how much effort,
polish, and rigour a task gets to the cost of getting it wrong and the value of
getting it right:

- **Throwaway script, prototype, internal note** — make it work, then stop.
- **Production code, security boundary, published artifact** — invest the rigour.

**NEVER** gold-plate low-stakes work, and **NEVER** ship high-stakes work at
low-stakes quality. When you cannot tell which a task is, ask before assuming.
